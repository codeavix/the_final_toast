The Final Toast
A terminal-based murder mystery game written in C++17.
Victor Cross — powerful film producer — is found dead at his own premiere after-party. Poison in his private whiskey decanter. The mansion is sealed. Four suspects. One killer.
You are the detective.
How to Play
Mac / Linux / WSL
g++ -std=c++17 -o final_toast main.cpp GameEngine.cpp
./final_toast
Windows (MinGW)
g++ -std=c++17 -o final_toast.exe main.cpp GameEngine.cpp
final_toast.exe
